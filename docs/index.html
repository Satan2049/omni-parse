How it works

A simple pipeline from input to RAG-ready artifacts.

01
Paste a URL or HTML Use the Next.js workspace — configure timeouts, user-agent, and JS rendering.
02
Fetch & extract httpx or Playwright fetches the page; Trafilatura extracts title, body, metadata, and images.
03
Preview & export Inspect Markdown, JSON, or text in the preview panel — download or convert via the API.

HTTP API

Integrate extraction into scripts and automation. Interactive docs at /docs when the server is running.

Method Path Description
POST /extract Extract title, markdown, metadata, and image URLs
POST /convert Convert text or markdown to PDF, TXT, or MD download
GET /health Health check for orchestration and the UI status badge 
# Example: extract from a URL
curl -X POST http://localhost:8000/extract \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/article", "render_js": false}'

Quick start (Windows)

Prerequisites: Python 3.11+ and Node.js 20+. Double-click start.bat in the repo root. 

# Services after launch
UI       →  http://localhost:3000
API      →  http://localhost:8000
API Docs →  http://localhost:8000/docs

Prefer manual setup? See the README for backend and frontend steps on macOS and Linux.
